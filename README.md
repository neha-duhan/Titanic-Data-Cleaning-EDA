# Titanic Data Cleaning and Exploratory Data Analysis

## Objective
Perform data cleaning and exploratory data analysis using Python.

## Tools Used
- Python
- Pandas
- NumPy
- KaggleHub
- Jupyter Notebook

## Steps Performed
1. Loaded Titanic dataset from Kaggle.
2. Identified missing values using `.isnull().sum()`.
3. Filled missing numerical values with the mean.
4. Removed duplicate rows.
5. Generated statistical summaries using `.describe()`.
6. Calculated:
   - Average Age
   - Highest Fare
   - Average Fare
   - Survival Count

## Dataset
Titanic Data for Data Preprocessing (Kaggle)
